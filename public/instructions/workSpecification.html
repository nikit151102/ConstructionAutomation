Инструкция по созданию ведомости объёмов работ

  1. Для создания ведомости объёмов работ Вам потребуется один файл:

    • КС-2 по выполненным работам (фактическая смета) форматом «xls» (эксель).
      КС-2 составляется сметчиком на основе Локальной сметы с фактически выполненными объёмами работ.

  2. Для грамотного составления ведомости объёмов работ, прошу проверить заполнение файла КС-2:

    • В «шапке» КС-2 должны быть заполнены строки:
      • Заказчик
      • Подрядчик
      • Стройка
      • Объект
      • Договор подряда: номер и дата

  3. При наличии заполненного нужным образом файла КС-2, Ваша ведомость объёмов работ будет готова в течение нескольких секунд.
